# Terminology

This project is being prepared as an English-first open-source project.

## Document Types

| Term | Meaning |
| --- | --- |
| Invoice | A billing or payment request document. |
| Contract | A legal or commercial agreement. |
| Receipt | A document confirming payment. |
| Delivery Note | A document recording shipment or delivery. |
| Payment Receipt | A document confirming money received. |

## Common Fields

| Term | Meaning |
| --- | --- |
| Buyer | Purchasing party. |
| Seller | Selling party. |
| Vendor | Supplier or service provider. |
| Counterparty | The other party in a contract or transaction. |
| Total Amount | Final document amount. |
| Document Date | Invoice date, contract signing date, or related date. |
| Confidence | Estimated extraction reliability. |
