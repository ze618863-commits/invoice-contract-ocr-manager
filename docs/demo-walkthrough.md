# Demo Walkthrough

Use synthetic data only. Do not use real invoices, contracts, company names, tax IDs, addresses, phone numbers, bank accounts, QR codes, or customer data.

## Suggested Demo Flow

1. Start the backend.
2. Start the frontend.
3. Upload a synthetic invoice or contract.
4. Review extracted OCR fields.
5. Correct uncertain fields.
6. Save metadata.
7. Confirm the document appears in the local archive workflow.
